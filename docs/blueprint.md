# **App Name**: GPU Recs

## Core Features:

- Requirement Input: Input fields for users to specify their requirements (budget, intended use, desired performance, etc.)
- Information Retrieval: Use the users inputs to tool retrieve information about available GPUs.
- Ranking & Selection: Rank available GPUs, and determine which one best matches user needs.
- Recommendation Display: Display the recommended GPU along with key specifications, pricing, and links to purchase.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), evoking a sense of technology, reliability, and trust.
- Background color: Light gray (#F5F5F5) for a clean and modern aesthetic.
- Accent color: Electric Purple (#8A2BE2) for highlights and call-to-action buttons.
- Body and headline font: 'Inter' sans-serif, offering a neutral, clean, and highly readable typeface for both headings and body text.
- Use minimalist icons related to performance metrics (FPS, resolution) and component types (GPU, CPU).
- Employ a card-based layout for displaying GPU recommendations, with clear sections for specifications and pricing.
